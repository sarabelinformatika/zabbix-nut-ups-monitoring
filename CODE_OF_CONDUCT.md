# Code of Conduct

## Our Commitment

We are committed to providing a welcoming, respectful and professional environment for everyone contributing to this project.

Participation in this project means agreeing to:

- Be respectful.
- Be constructive.
- Be professional.
- Help improve the project.
- Respect different opinions and technical approaches.

## Unacceptable Behavior

The following will not be tolerated:

- Harassment
- Personal attacks
- Hate speech
- Spam
- Offensive language
- Malicious contributions

## Contributions

Constructive feedback, documentation improvements, bug reports and feature requests are always welcome.

## Enforcement

Project maintainers reserve the right to edit, reject or remove contributions that violate this Code of Conduct.
