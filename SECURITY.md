# Security Policy

## Supported Versions

Only the latest stable release is actively maintained.

## Reporting a Vulnerability

Please do not publicly disclose security vulnerabilities before contacting the maintainers.

Report security issues through GitHub Security Advisories or contact:

https://sarabelinformatika.hu

Please include:

- Product version
- Operating system
- Steps to reproduce
- Expected behaviour
- Actual behaviour
- Relevant logs (if available)

Security reports will be reviewed as quickly as possible.
