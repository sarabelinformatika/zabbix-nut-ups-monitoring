# Contributing

Thank you for your interest in improving this project.

We welcome:

- Bug reports
- Documentation improvements
- Feature suggestions
- Pull Requests

## Before opening an Issue

Please verify:

- The issue has not already been reported.
- You are using a supported version.
- The documentation has been reviewed.

## Pull Requests

Please ensure that:

- Documentation is updated where applicable.
- Configuration examples are tested.
- Changes remain compatible with supported platforms.
- Code follows existing project style.

## Documentation

Documentation improvements are always appreciated.

## Questions

If you have questions regarding the project, please open a GitHub Issue.

Thank you for contributing.
